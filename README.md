# hexo-gitalk-init

Hexo gitalk 轻量级初始化工具

## Feature

1. 无需安装其他的依赖，让你的hexo项目更干净
2. 不需要依赖或修改sitemap
3. 更适合自动化
4. 代码强迫症福音

## 使用示例

### 

```json5
{
  "username": "GITHUB_USERNAME",
  "repo": "GITHUB_REPO",
  "token": "GITALK_TOKEN",
  "cache": true,
  "outputCacheFile": "./public/gitalk-init-cache.json",
  "outputCacheRemote": "OUTPUT_CACHE_REMOTE"
}
```
